# Design vs Production — Decision Tool

An interactive decision tree to help Delivery PMs route work to the right team.

## Deploy to Vercel

### Option A — GitHub (recommended)
1. Push this folder to a GitHub repo
2. Go to [vercel.com](https://vercel.com) → Add New Project
3. Import your GitHub repo
4. Vercel auto-detects Vite — just click **Deploy**

### Option B — Vercel CLI
```bash
npm install -g vercel
npm install
vercel
```

## Run locally
```bash
npm install
npm run dev
```
